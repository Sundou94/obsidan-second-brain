# 🧠 Core Concepts

**AI-RAN**  
* 인공지능 기반의 무선 액세스 네트워크  
* 통신망의 효율성과 성능을 극대화할 수 있게 해줌.  

**vRAN**  
* 가상화된 무선 액세스 네트워크  
* 하드웨어를 최소화하고 유연성을 높여줌.  

**Massive MIMO**  
* 다수의 안테나를 사용하여 통신 효율을 높이는 기술  
* 데이터 전송 속도가 향상되어 기지국 수를 줄일 수 있음.  

**빔포밍**  
* 신호를 특정 방향으로 집중시키는 기술  
* 통신의 품질을 높이고 간섭을 줄여줌.

# 🔑 Key Terms

* [[AI-RAN]] : 인공지능 기반의 무선 액세스 네트워크로 성능을 향상시킴.
* [[vRAN]] : 가상화된 무선 네트워크, 효율적인 운영 방식을 제공.
* [[Massive MIMO]] : 다수의 안테나를 이용해 통신 성능을 극대화.
* [[빔포밍]] : 신호 전달을 최적화하여 품질과 속도를 개선.
* [[6G]] : 차세대 통신 기술로 기지국의 진화 설명.

- 통신 기지국이 인공지능 데이터센터 역할을 하게 되면서, 무선 네트워크의 구조적 변화가 필요함. 
- AI와 통신 인프라의 통합이 신속한 데이터 처리와 응답 시간 단축에 기여.
- Massive MIMO와 빔포밍 같은 최신 기술을 통합해야 효율적인 네트워크를 구축할 수 있음.
