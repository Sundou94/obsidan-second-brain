[05_몬테카를로_방법_필기]

몬테 카를로 방법 이란 ?
반복적인 무작위 샘플링에 의존하는 계산 알고리즘 이다.
핵심 아이디어는 : 직접 계산하기 어려운 값을 수많은 무작위 실험을 통해 추정하는 방법 이다. 


주사위 두개의 합
주사위 두개를 던져서 나오는 합의 확률 분포
==기댓값 이란 무수히 많이 시행했을때 나오는 평균 값을 의미한다.==
분포 모델  vs  샘플 모델

분포모델은 
주사위를 확률표로 계산 하는 것이다.

샘플 모델
실제 주사위를 던져서 평균을 계산 하는 것이다. 

증분 업데이트 수식
몬테카를로 방법으로 기댓값 구하는 두가지 방식
