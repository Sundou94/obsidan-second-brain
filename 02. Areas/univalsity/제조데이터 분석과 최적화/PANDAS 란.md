## **3️⃣ 전처리 (너 코드 핵심🔥)**

```
df.dropna()
df.drop_duplicates()
df.fillna(0)
```



 당신이 제공한 코드 조각은 pandas를 사용하여 데이터 전처리를 수행하는 예제입니다. 각각의 기능을 짧게 설명하자면:

1. **`df = df_raw.copy()`**: 원본 데이터프레임 `df_raw`를 복사하여 `df`라는 새로운 데이터프레임을 만듭니다. 이는 원본 데이터를 안전하게 보존하면서 데이터를 조작할 수 있게 해줍니다.

2. **`df.drop_duplicates()`**: 중복된 행을 제거합니다. 데이터셋에서 중복된 정보를 정리하여 분석의 정확성을 높입니다.

3. **`df.dropna()`**: 결측값(NaN)이 있는 행을 삭제합니다. 이는 분석에 방해가 될 수 있는 누락된 데이터를 제거하여 데이터의 일관성을 유지합니다.

4. **`df.groupby()`**: 특정 컬럼을 기준으로 데이터를 그룹화하여 각 그룹에 대한 통계나 집계 작업을 수행할 수 있게 합니다. 예를 들어, 같은 카테고리의 평균값이나 합계를 계산하는 데 유용합니다.
