# Quiz Q3 풀이 — 4-Grid World Bellman Equation

> **문제**: 2-Grid World를 4-Grid World로 확장할 때, state value에 대한 Bellman equation을 적고, 각 state value를 구하라.

---

## 1. 환경 설정 (Environment Setup)

```
왼쪽 벽                                     오른쪽 벽
  -1   | L1 | L2 | L3 | L4 |   -1
 (벽)   [🤖]             [🍎]  (벽)
                        +1
                    (L3→L4 전이 시)
```

| 항목 | 설정 |
|------|------|
| 상태 | $S = \{L1, L2, L3, L4\}$ |
| 행동 | $A = \{\text{LEFT, RIGHT}\}$ |
| 정책 | $\pi(\text{LEFT}\|s) = \pi(\text{RIGHT}\|s) = 0.5$ (모든 상태) |
| 할인율 | $\gamma = 0.9$ |

---

## 2. 상태 전이 및 보상 (Transition & Reward)

| 현재 상태 | 행동 | 다음 상태 | 보상 $r$ |
|-----------|------|-----------|----------|
| L1 | LEFT | **L1** (벽) | **-1** |
| L1 | RIGHT | L2 | 0 |
| L2 | LEFT | L1 | 0 |
| L2 | RIGHT | L3 | 0 |
| L3 | LEFT | L2 | 0 |
| L3 | RIGHT | **L4** (사과) | **+1** |
| L4 | LEFT | L3 | 0 |
| L4 | RIGHT | **L4** (벽) | **-1** |

> **2-Grid World와의 비교**: 2-Grid에서는 $L1 \to L1$ (LEFT, -1), $L1 \to L2$ (RIGHT, +1), $L2 \to L1$ (LEFT, 0), $L2 \to L2$ (RIGHT, -1)이었음. 4-Grid는 중간 셀(L2, L3)이 추가되고 사과(+1)가 L4로 이동.

---

## 3. Bellman Equation 도출

Bellman Equation 일반식:

$$v_\pi(s) = \sum_a \pi(a|s) \left\{ r(s,a,s') + \gamma v_\pi(s') \right\}$$

---

### 상태 L1

$$v_\pi(L1) = 0.5\{-1 + 0.9\,v_\pi(L1)\} + 0.5\{0 + 0.9\,v_\pi(L2)\}$$

$$v_\pi(L1) = -0.5 + 0.45\,v_\pi(L1) + 0.45\,v_\pi(L2)$$

$$\boxed{0.55\,v_\pi(L1) - 0.45\,v_\pi(L2) = -0.5} \quad \cdots (1)$$

---

### 상태 L2

$$v_\pi(L2) = 0.5\{0 + 0.9\,v_\pi(L1)\} + 0.5\{0 + 0.9\,v_\pi(L3)\}$$

$$v_\pi(L2) = 0.45\,v_\pi(L1) + 0.45\,v_\pi(L3)$$

$$\boxed{-0.45\,v_\pi(L1) + v_\pi(L2) - 0.45\,v_\pi(L3) = 0} \quad \cdots (2)$$

---

### 상태 L3

$$v_\pi(L3) = 0.5\{0 + 0.9\,v_\pi(L2)\} + 0.5\{1 + 0.9\,v_\pi(L4)\}$$

$$v_\pi(L3) = 0.45\,v_\pi(L2) + 0.5 + 0.45\,v_\pi(L4)$$

$$\boxed{-0.45\,v_\pi(L2) + v_\pi(L3) - 0.45\,v_\pi(L4) = 0.5} \quad \cdots (3)$$

---

### 상태 L4

$$v_\pi(L4) = 0.5\{0 + 0.9\,v_\pi(L3)\} + 0.5\{-1 + 0.9\,v_\pi(L4)\}$$

$$v_\pi(L4) = 0.45\,v_\pi(L3) - 0.5 + 0.45\,v_\pi(L4)$$

$$\boxed{-0.45\,v_\pi(L3) + 0.55\,v_\pi(L4) = -0.5} \quad \cdots (4)$$

---

## 4. 연립방정식 정리

4개의 미지수 $\{v_\pi(L1),\ v_\pi(L2),\ v_\pi(L3),\ v_\pi(L4)\}$에 대한 1차 연립방정식:

$$\begin{cases}
0.55\,v_\pi(L1) - 0.45\,v_\pi(L2) \phantom{- 0.45\,v_\pi(L3) - 0.45\,v_\pi(L4)} = -0.5 \quad (1)\\
-0.45\,v_\pi(L1) + v_\pi(L2) - 0.45\,v_\pi(L3) \phantom{- 0.45\,v_\pi(L4)} = \phantom{-}0 \quad (2)\\
\phantom{-0.45\,v_\pi(L1)} -0.45\,v_\pi(L2) + v_\pi(L3) - 0.45\,v_\pi(L4) = \phantom{-}0.5 \quad (3)\\
\phantom{-0.45\,v_\pi(L1) + v_\pi(L2)} -0.45\,v_\pi(L3) + 0.55\,v_\pi(L4) = -0.5 \quad (4)
\end{cases}$$

행렬 형식으로 표현하면:

$$\begin{bmatrix} 0.55 & -0.45 & 0 & 0 \\ -0.45 & 1 & -0.45 & 0 \\ 0 & -0.45 & 1 & -0.45 \\ 0 & 0 & -0.45 & 0.55 \end{bmatrix} \begin{bmatrix} v(L1) \\ v(L2) \\ v(L3) \\ v(L4) \end{bmatrix} = \begin{bmatrix} -0.5 \\ 0 \\ 0.5 \\ -0.5 \end{bmatrix}$$

---

## 5. 풀이 과정

### Step 1: (1)에서 $v(L1)$ 표현

$$v(L1) = \frac{-0.5 + 0.45\,v(L2)}{0.55} \quad \cdots (I)$$

### Step 2: (4)에서 $v(L4)$ 표현

$$v(L4) = \frac{-0.5 + 0.45\,v(L3)}{0.55} \quad \cdots (IV)$$

### Step 3: (I)을 (2)에 대입 → $v(L2), v(L3)$에 대한 식 도출

$$-0.45 \cdot \frac{-0.5 + 0.45\,v(L2)}{0.55} + v(L2) - 0.45\,v(L3) = 0$$

$$\frac{0.225 - 0.2025\,v(L2)}{0.55} + v(L2) - 0.45\,v(L3) = 0$$

$$0.4091 + 0.6318\,v(L2) - 0.45\,v(L3) = 0 \quad \cdots (II')$$

### Step 4: (IV)를 (3)에 대입

$$-0.45\,v(L2) + v(L3) - 0.45 \cdot \frac{-0.5 + 0.45\,v(L3)}{0.55} = 0.5$$

$$-0.45\,v(L2) + 0.6318\,v(L3) = 0.0909 \quad \cdots (III')$$

### Step 5: (II'), (III') 연립 풀이

$$\begin{cases} 0.6318\,v(L2) - 0.45\,v(L3) = -0.4091 \\ -0.45\,v(L2) + 0.6318\,v(L3) = 0.0909 \end{cases}$$

(II') × 0.6318 + (III') × 0.45 로 소거 후 풀면:

$$v(L2) \approx -1.1061, \quad v(L3) \approx -0.6439$$

### Step 6: $v(L1)$, $v(L4)$ 역대입

$$v(L1) = \frac{-0.5 + 0.45 \times (-1.1061)}{0.55} \approx -1.8141$$

$$v(L4) = \frac{-0.5 + 0.45 \times (-0.6439)}{0.55} \approx -1.4359$$

---

## 6. 최종 결과

$$\begin{cases}
v_\pi(L1) = -1.8141 \\
v_\pi(L2) = -1.1061 \\
v_\pi(L3) = -0.6439 \\
v_\pi(L4) = -1.4359
\end{cases}$$

```
 상태:   L1       L2       L3       L4
 값:   -1.8141  -1.1061  -0.6439  -1.4359
         ↑                         ↑
      (가장 낮음)              (두 번째로 낮음)
```

---

## 7. 결과 해석

| 상태 | State Value | 해석 |
|------|------------|------|
| L1 | **-1.8141** | 왼쪽 벽에 가장 가까워 벽 충돌(-1) 위험이 높음 → 가치가 가장 낮음 |
| L2 | -1.1061 | L1보다는 낫지만 여전히 사과(L4)에서 멀리 있음 |
| L3 | **-0.6439** | 사과(L4) 바로 왼쪽. +1 보상에 가장 가깝지만 오른쪽 벽(-1)의 영향도 받음 → **가치가 가장 높음** |
| L4 | -1.4359 | 사과(+1)를 얻지만 오른쪽 벽 충돌(-1) 위험이 높고, 왼쪽 이동 시 다시 L3로 돌아가는 비효율 발생 |

> **핵심 관찰**: 모든 state value가 음수인 이유는 stochastic policy(0.5/0.5)가 최적이 아니기 때문. 벽 충돌(-1)이 자주 발생하여 평균적으로 음의 수익을 얻게 됨.
