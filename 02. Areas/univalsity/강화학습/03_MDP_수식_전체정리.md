## 1. 기본 기호 정의 (p.2)

> 📄 **원본 슬라이드**: p.2

---

### 수식 1-1. 상태 (State)

$$S_t \in \mathcal{S}$$

| 기호 | 의미 |
|------|------|
| $S_t$ | 시각 $t$에서 에이전트가 관찰하는 **현재 상태** |
| $\mathcal{S}$ | 가능한 모든 상태의 집합 (State Space) |
| $\in$ | "~에 속한다" — $S_t$는 $\mathcal{S}$ 안의 원소 중 하나 |

> **수식의 의미**: 시각 t에서 에이전트의 상태 $S_t$는 가능한 상태 집합 $\mathcal{S}$ 안에 있다. 예) Grid World에서 $\mathcal{S} = \{(0,0), (0,1), \ldots\}$ (각 셀의 좌표들)

---

### 수식 1-2. 행동 (Action)

$$A_t \in \mathcal{A}(S_t)$$

| 기호 | 의미 |
|------|------|
| $A_t$ | 시각 $t$에서 에이전트가 선택한 **행동** |
| $\mathcal{A}(S_t)$ | 상태 $S_t$에서 선택 가능한 **행동의 집합** |

> **수식의 의미**: 에이전트는 현재 상태 $S_t$에서 취할 수 있는 행동 집합 $\mathcal{A}(S_t)$ 중 하나를 선택한다. 상태마다 가능한 행동이 다를 수 있다. 예) Recycling Robot에서 배터리가 high일 때 $\mathcal{A}(\text{high}) = \{\text{search, wait}\}$

---

### 수식 1-3. 보상 (Reward)

$$R_{t+1} \in \mathcal{R} \subset \mathbb{R}$$

| 기호 | 의미 |
|------|------|
| $R_{t+1}$ | 시각 $t$에서 행동 $A_t$를 취한 **직후** 받는 보상 |
| $\mathcal{R}$ | 가능한 보상값들의 집합 |
| $\mathbb{R}$ | 실수 전체 집합 |
| $\subset$ | "~의 부분집합" — 보상은 실수 중 일부 값만 가짐 |

> **수식의 의미**: 시각 $t$에 행동을 취하면 다음 시각 $t+1$에 보상 $R_{t+1}$을 받는다. 보상은 실수 집합의 부분집합에서 나온다. **주의**: $R_{t+1}$인 이유는 행동 $A_t$의 결과가 한 스텝 뒤에 반영되기 때문이다.

---

### 수식 1-4. 정책 (Policy)

$$\pi_t = \pi_t(a|s)$$

| 기호 | 의미 |
|------|------|
| $\pi_t$ | 시각 $t$에서의 **정책** (policy) |
| $\pi_t(a\|s)$ | 상태 $s$일 때 행동 $a$를 선택할 **확률** |
| $a$ | 특정 행동 값 |
| $s$ | 특정 상태 값 |

> **수식의 의미**: 정책 $\pi$는 각 상태에서 어떤 행동을 얼마나 자주 선택할지를 정의한다.
> - **결정론적 정책 (deterministic)**: $\pi(a|s) = 1$ (항상 특정 행동만 선택)
> - **확률적 정책 (stochastic)**: $0 < \pi(a|s) < 1$ (여러 행동을 확률로 선택)
> - RL의 목표: 총 보상을 최대화하는 정책 $\pi$를 찾는 것

## 2. Return 수식 (p.6)

> 📄 **원본 슬라이드**: p.6

---

### 수식 2-1. Episodic Task의 Return

$$G_t = R_{t+1} + R_{t+2} + R_{t+3} + \cdots + R_T$$

| 기호 | 의미 |
|------|------|
| $G_t$ | 시각 $t$ 이후 받는 **누적 보상 (Return / 수익)** |
| $R_{t+1}, R_{t+2}, \ldots$ | 각 시각에서 받는 개별 보상 |
| $T$ | **Terminal state에 도달하는 마지막 시각** (에피소드 종료 시점) |

> **수식의 의미**: Episodic task(끝이 있는 과제)에서 Return은 현재 시각 $t$ 이후부터 에피소드가 끝날 때까지 받는 모든 보상의 단순 합이다. 할인 없이 미래 보상을 동등하게 취급한다.
> 예) 게임에서 이길 때까지 얻는 점수의 합

---

### 수식 2-2. Continuing Task의 Return (할인된 Return)

$$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

| 기호 | 의미 |
|------|------|
| $G_t$ | 시각 $t$ 이후 받는 **할인된 누적 보상** |
| $\gamma$ | **할인율 (discount rate)**, $0 \leq \gamma \leq 1$ |
| $\gamma^k$ | $k$ 스텝 이후 보상에 적용되는 할인 계수 |
| $k$ | 현재 시각으로부터의 미래 스텝 수 |
| $R_{t+k+1}$ | 현재로부터 $k+1$ 스텝 후에 받는 보상 |

> **수식의 의미**: 끝이 없는 task에서는 미래 보상을 현재 가치로 할인하여 합산한다. $\gamma$가 작을수록 먼 미래의 보상을 덜 중요하게 여기고, $\gamma = 1$이면 모든 미래 보상을 동등하게 취급한다 (이 경우 $G_t \to \infty$ 가능성 있음).
>
> **할인율의 직관**: "지금 당장 받는 1원이 1년 후 받는 1원보다 가치 있다" — 마치 은행 이자처럼 미래 가치를 현재 가치로 환산하는 개념

---

### 수식 2-3. Return의 재귀 관계

$$G_t = R_{t+1} + \gamma G_{t+1}$$

| 기호 | 의미 |
|------|------|
| $G_t$ | 현재 시각 $t$에서의 Return |
| $R_{t+1}$ | 다음 스텝에서 받는 즉각 보상 |
| $\gamma$ | 할인율 |
| $G_{t+1}$ | 다음 시각 $t+1$에서의 Return |

> **수식의 의미**: Return은 "지금 당장 받는 보상 + 다음 시각의 Return을 할인한 값"으로 재귀적으로 표현된다. 이 관계가 **Bellman Equation의 핵심 구조**다.

## 3. Markov Property (p.7)

> 📄 **원본 슬라이드**: p.7

---

### 수식 3-1. Markov 성질 정의

$$\Pr\{S_{t+1} \mid S_0, S_1, \cdots, S_t\} = \Pr\{S_{t+1} \mid S_t\}$$

| 기호 | 의미 |
|------|------|
| $\Pr\{\cdot\}$ | 확률 (Probability) |
| $S_{t+1}$ | 다음 상태 |
| $S_0, S_1, \cdots, S_t$ | 처음부터 현재까지의 **모든 상태 이력 (history)** |
| $S_t$ | **현재 상태만** |
| $\mid$ | 조건부 확률 기호 ("~가 주어졌을 때") |

> **수식의 의미**: Markov 성질 — 다음 상태를 예측할 때, 과거 전체 이력을 알더라도 **현재 상태만 알면 충분**하다. 현재 상태가 미래를 결정하는 데 필요한 모든 정보를 담고 있다.
> "현재가 주어지면, 미래는 과거와 독립이다 (The future is independent of the past given the present)"

---

### 수식 3-2. 강화학습에서의 Markov 성질 적용

$$\Pr\{R_{t+1} = r,\ S_{t+1} = s' \mid S_0, A_0, R_1, \cdots, S_{t-1}, A_{t-1}, R_t, S_t, A_t\}$$
$$= \Pr\{R_{t+1} = r,\ S_{t+1} = s' \mid S_t, A_t\} \quad \text{(Markov property)}$$
$$= p(s', r \mid s, a)$$

| 기호 | 의미 |
|------|------|
| $\Pr\{R_{t+1} = r,\ S_{t+1} = s' \mid \cdots\}$ | 다음 보상이 $r$이고 다음 상태가 $s'$일 조건부 확률 |
| 좌변의 조건 | 처음부터 현재까지의 **모든 상태·행동·보상 이력** |
| $S_t, A_t$ | **현재 상태와 행동만** |
| $p(s', r \mid s, a)$ | MDP의 **전이 다이나믹스 함수** (transition dynamics) |

> **수식의 의미**: RL에서 Markov 성질은 "다음 상태 $s'$와 보상 $r$의 확률은 전체 과거 이력이 아닌 **현재 상태 $s$와 행동 $a$만으로 결정**된다"는 것이다. 이 함수 $p(s', r|s, a)$가 환경 모델의 핵심이다.

## 4. MDP 전이 확률 및 기댓값 수식 (p.8)

> 📄 **원본 슬라이드**: p.8

MDP는 튜플 $(\mathcal{S}, \mathcal{A}, T, r, \gamma)$로 정의된다.

---

### 수식 4-1. 전이 다이나믹스 함수 (One-step Dynamics)

$$p(s', r \mid s, a) = \Pr\{S_{t+1} = s',\ R_{t+1} = r \mid S_t = s,\ A_t = a\}$$

| 기호 | 의미 |
|------|------|
| $p(s', r \mid s, a)$ | 환경의 **전이 다이나믹스** 함수 |
| $s'$ | 다음 상태 (next state) |
| $r$ | 다음 스텝에서 받는 보상 값 |
| $s$ | 현재 상태 (current state) |
| $a$ | 현재 행동 (current action) |
| $\Pr\{\cdot\}$ | 확률 |

> **수식의 의미**: 현재 상태가 $s$이고 행동 $a$를 취했을 때, 다음 상태가 $s'$가 되고 보상 $r$을 받을 **결합 확률**이다. 이 함수 하나로 환경의 모든 동역학을 표현한다. MDP에서는 이 확률이 오직 $(s, a)$에만 의존한다 (Markov 성질).

---

### 수식 4-2. 상태 전이 확률 (State-Transition Probability)

$$p(s' \mid s, a) = \Pr\{S_{t+1} = s' \mid S_t = s,\ A_t = a\} = \sum_{r \in \mathcal{R}} p(s', r \mid s, a)$$

| 기호 | 의미 |
|------|------|
| $p(s' \mid s, a)$ | 상태 전이 확률 (T 행렬의 원소) |
| $\sum_{r \in \mathcal{R}}$ | 가능한 모든 보상 $r$에 대해 합산 |

> **수식의 의미**: 보상값에 관계없이, 상태 $s$에서 행동 $a$를 취했을 때 **다음 상태가 $s'$이 될 확률**이다. $p(s', r|s, a)$에서 보상 $r$을 모두 더해(주변화/marginalize) 구한다.
>
> 모든 가능한 $s'$에 대해 합산하면 1이 된다: $\sum_{s'} p(s'|s, a) = 1$

---

### 수식 4-3. 상태-행동 쌍에 대한 기대 보상

$$r(s, a) = \mathbb{E}[R_{t+1} \mid S_t = s,\ A_t = a] = \sum_{r \in \mathcal{R}} r \sum_{s' \in \mathcal{S}} p(s', r \mid s, a)$$

| 기호 | 의미 |
|------|------|
| $r(s, a)$ | 상태 $s$에서 행동 $a$를 취했을 때의 **기대 보상** |
| $\mathbb{E}[\cdot]$ | 기댓값 (Expected value) |
| $\sum_{r \in \mathcal{R}} r \cdot (\cdots)$ | 각 보상 $r$에 그 확률을 곱해 합산 |
| $\sum_{s' \in \mathcal{S}} p(s', r \mid s, a)$ | 다음 상태를 모두 합산하여 $(s,a,r)$의 주변 확률 계산 |

> **수식의 의미**: 현재 상태 $s$에서 행동 $a$를 취할 때, **평균적으로 받을 것으로 기대되는 보상**이다. 모든 가능한 보상 $r$과 다음 상태 $s'$에 대해 확률을 가중치로 합산한다.

---

### 수식 4-4. 상태-행동-다음상태 트리플에 대한 기대 보상

$$r(s, a, s') = \mathbb{E}[R_{t+1} \mid S_t = s,\ A_t = a,\ S_{t+1} = s'] = \frac{\sum_{r \in \mathcal{R}} r \cdot p(s', r \mid s, a)}{p(s' \mid s, a)}$$

| 기호 | 의미 |
|------|------|
| $r(s, a, s')$ | $(s, a, s')$ 트리플에서의 **기대 보상** |
| $\mathbb{E}[R_{t+1} \mid S_t=s, A_t=a, S_{t+1}=s']$ | $s$에서 $a$를 취해 $s'$로 전이될 때의 기대 보상 |
| 분자 $\sum_{r} r \cdot p(s', r \mid s, a)$ | 다음 상태가 $s'$인 조건 하에서 보상의 기댓값 계산을 위한 합 |
| 분모 $p(s' \mid s, a)$ | **정규화** — 조건부 확률 $P(\text{보상}=r \mid s, a, s')$ 형성 |

> **수식의 의미**: 상태 $s$에서 행동 $a$를 해서 $s'$로 이동했을 때 **기대되는 보상**이다. $r(s,a)$가 다음 상태를 고려하지 않는 것과 달리, 이 수식은 다음 상태 $s'$까지 조건으로 고정한 더 세밀한 버전이다.
>
> **4가지 수식 관계 정리**:
> - $p(s',r|s,a)$: 가장 기본적인 joint 확률 (환경의 완전한 정보)
> - $p(s'|s,a)$: $p(s',r|s,a)$를 $r$로 marginalize
> - $r(s,a)$: $p(s',r|s,a)$를 $s'$와 $r$로 기댓값 계산
> - $r(s,a,s')$: $p(s',r|s,a)$를 $s'$ 고정 후 $r$로 기댓값 계산

## 5. Value Function 수식 (p.12)

> 📄 **원본 슬라이드**: p.12

---

### 수식 5-1. 상태 가치 함수 (State-Value Function)

$$v_\pi(s) = \mathbb{E}_\pi[G_t \mid S_t = s] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\Bigg|\; S_t = s\right]$$

$$v_\pi : \mathcal{S} \to \mathbb{R}$$

| 기호 | 의미 |
|------|------|
| $v_\pi(s)$ | 정책 $\pi$ 하에서 상태 $s$의 **가치** |
| $\mathbb{E}_\pi[\cdot]$ | 정책 $\pi$를 따를 때의 **기댓값** |
| $G_t$ | 시각 $t$ 이후의 (할인된) 누적 보상 |
| $S_t = s$ | 현재 상태가 $s$임을 조건으로 함 |
| $\sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$ | 미래 보상의 할인 합 (Return의 전개식) |
| $v_\pi : \mathcal{S} \to \mathbb{R}$ | 상태 공간을 실수로 매핑하는 함수 |

> **수식의 의미**: 정책 $\pi$를 따를 때, **상태 $s$에서 출발하여 앞으로 받을 것으로 기대되는 누적 보상의 기댓값**이다. 이 값이 클수록 그 상태가 "좋은 상태"임을 뜻한다. 에이전트는 높은 가치 함수를 가진 상태로 이동하려 한다.

---

### 수식 5-2. 행동 가치 함수 (Action-Value Function)

$$q_\pi(s, a) = \mathbb{E}_\pi[G_t \mid S_t = s,\ A_t = a] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\Bigg|\; S_t = s,\ A_t = a\right]$$

$$q_\pi : \mathcal{S} \times \mathcal{A} \to \mathbb{R}$$

| 기호 | 의미 |
|------|------|
| $q_\pi(s, a)$ | 정책 $\pi$ 하에서 상태 $s$에서 행동 $a$를 취할 때의 **가치** |
| $\mathbb{E}_\pi[\cdot]$ | 정책 $\pi$를 따를 때의 기댓값 |
| $S_t = s,\ A_t = a$ | 현재 상태 $s$에서 행동 $a$를 선택함을 조건으로 함 |
| $\mathcal{S} \times \mathcal{A}$ | 상태-행동 쌍의 공간 |

> **수식의 의미**: 정책 $\pi$를 따를 때, **상태 $s$에서 행동 $a$를 취한 후 앞으로 받을 기대 누적 보상**이다. $v_\pi(s)$가 상태의 좋고 나쁨을 나타내는 반면, $q_\pi(s,a)$는 **특정 행동의 좋고 나쁨**을 나타낸다.
>
> **$v_\pi$와 $q_\pi$의 관계**:
> $$v_\pi(s) = \sum_a \pi(a|s) \cdot q_\pi(s, a)$$
> 상태의 가치는 그 상태에서 취할 수 있는 모든 행동의 가치를 정책 확률로 가중 평균한 것이다.

## 6. Optimal Value & Policy 수식 (p.13)

> 📄 **원본 슬라이드**: p.13

---

### 수식 6-1. 최적 상태 가치 함수

$$v_*(s) = \max_\pi v_\pi(s)$$

| 기호 | 의미 |
|------|------|
| $v_*(s)$ | 상태 $s$에서의 **최적 상태 가치** |
| $\max_\pi$ | 모든 가능한 정책 $\pi$ 중에서 **최댓값** |
| $v_\pi(s)$ | 정책 $\pi$ 하에서 상태 $s$의 가치 |

> **수식의 의미**: 어떤 상태 $s$에 대해 **존재하는 모든 정책 중 최선의 정책을 따랐을 때 얻을 수 있는 최대 가치**다. 이 값은 모든 정책 중 최고의 성능을 나타내는 기준점이다.

---

### 수식 6-2. 최적 행동 가치 함수

$$q_*(s, a) = \max_\pi q_\pi(s, a)$$

| 기호 | 의미 |
|------|------|
| $q_*(s, a)$ | 상태 $s$에서 행동 $a$를 취할 때의 **최적 행동 가치** |
| $\max_\pi$ | 모든 정책 중 최댓값 |

> **수식의 의미**: 상태 $s$에서 행동 $a$를 취한 뒤 **최선의 정책을 따랐을 때 기대되는 최대 누적 보상**이다. Q-Learning에서 직접 학습하는 대상이다.
>
> **$v_*$와 $q_*$의 관계**:
> $$v_*(s) = \max_a q_*(s, a)$$
> 최적 상태 가치 = 그 상태에서 선택 가능한 행동들 중 **최대 행동 가치**

---

### 수식 6-3. 최적 정책 (확률적 정책 버전)

$$\pi_*(a|s) > 0 \quad \text{only where} \quad q_*(s, a) = \max_b q_*(s, b), \quad \forall s \in \mathcal{S}$$

| 기호 | 의미 |
|------|------|
| $\pi_*(a\|s)$ | 최적 정책에서 상태 $s$일 때 행동 $a$를 선택할 확률 |
| $\max_b q_*(s, b)$ | 상태 $s$에서 모든 행동 $b$ 중 최대 $q_*$ 값 |
| $\forall s \in \mathcal{S}$ | 모든 상태 $s$에 대해 |

> **수식의 의미**: 최적 정책은 **$q_*(s,a)$가 최대인 행동에만 양의 확률을 부여**한다. 최적이 아닌 행동은 절대 선택하지 않는다. 최대 $q_*$를 갖는 행동이 여러 개라면 그들 사이에 확률을 분배할 수 있다.

---

### 수식 6-4. 최적 정책 (결정론적 정책 버전)

$$\pi_*(a|s) = \begin{cases} 1, & \text{if } a = \arg\max_{a \in \mathcal{A}} q_*(s, a) \\ 0, & \text{otherwise} \end{cases}$$

| 기호 | 의미 |
|------|------|
| $\arg\max_{a \in \mathcal{A}} q_*(s, a)$ | $q_*(s,a)$를 최대화하는 **행동 $a$** 자체 (값이 아닌 인덱스) |
| $1$ | 해당 행동을 **반드시** 선택 (확률 = 1) |
| $0$ | 해당 행동을 **절대** 선택하지 않음 (확률 = 0) |

> **수식의 의미**: 결정론적 최적 정책 — 각 상태에서 $q_*(s,a)$가 가장 큰 행동 하나만 선택한다. 가장 자주 쓰이는 형태의 최적 정책이다.

---

### 수식 6-5. 최적 행동 (Optimal Action)

$$\mu_*(s) = \arg\max_a q_*(s, a)$$

| 기호 | 의미 |
|------|------|
| $\mu_*(s)$ | 상태 $s$에서의 **최적 행동** |
| $\arg\max_a$ | $q_*(s,a)$를 최대화하는 행동 $a$의 **값** (확률이 아닌 행동 자체) |

> **수식의 의미**: 상태 $s$에서 취해야 할 최선의 행동을 바로 돌려주는 함수. 결정론적 정책의 압축 표현이다. $\pi_*(a|s)$가 모든 행동에 대한 확률 분포를 표현한다면, $\mu_*(s)$는 그 중 선택되는 행동 하나를 바로 반환한다.

## 7. Expectation 수식 (p.15)

### 수식 7-1. 기댓값 (Expectation)

$$\mathbb{E}[r(x, y)] = \sum_x \sum_y p(x, y)\, r(x, y) = \sum_x \sum_y p(x)\, p(y|x)\, r(x, y)$$

| 기호 | 의미 |
|------|------|
| $\mathbb{E}[\cdot]$ | **기댓값 (Expected value)** — 가능한 모든 경우에 대해 확률로 가중 평균 |
| $r(x, y)$ | 변수 $x$와 $y$의 조합에서 얻는 **보상(또는 임의의 함수값)** |
| $p(x, y)$ | $(x, y)$의 **결합 확률 (joint probability)** |
| $p(x)$ | $x$의 **주변 확률 (marginal probability)** |
| $p(y\|x)$ | $x$가 주어졌을 때 $y$의 **조건부 확률 (conditional probability)** |
| $\sum_x \sum_y$ | 가능한 모든 $(x, y)$ 쌍에 대해 합산 |

> **수식의 의미**: 기댓값은 각 결과에 그 결과가 일어날 확률을 곱해 모두 더한 값이다. 결합 확률 $p(x,y)$는 연쇄 법칙(chain rule)으로 $p(x) \cdot p(y|x)$로 분해 가능하다.
>
> **예시 (p.15, 주사위-동전)**: 주사위 $x \in \{1,\ldots,6\}$, 동전 $y \in \{0(\text{뒤}), 1(\text{앞})\}$일 때
> $$p(x=4, y=\text{앞}) = \frac{1}{6} \cdot \frac{4}{5} = \frac{4}{30}$$

## 8. 예제: 1x2 Grid World 가치 계산 (p.16)

> 📄 **원본 슬라이드**: p.16

---

### 수식 8-1. 결정론적 정책 $\pi_1$ (항상 right)에서의 상태 가치

$$v_{\mu_1}(S = L1) = 1 + 0.9 \cdot (-1) + 0.9^2 \cdot (-1) + \cdots$$

$$= 1 - 0.9(1 + 0.9 + 0.9^2 + \cdots)$$

$$= 1 - \frac{0.9}{1 - 0.9} = 1 - 9 = -8$$

| 기호 | 의미 |
|------|------|
| $v_{\mu_1}(S = L1)$ | 정책 $\mu_1$ (항상 right) 하에서 상태 L1의 가치 |
| $1$ | L1 → L2 이동 시 첫 번째 보상 (+1, 사과 획득) |
| $0.9^k \cdot (-1)$ | 이후 벽에 부딪힐 때마다 받는 보상 (-1)의 할인값 |
| $\frac{0.9}{1-0.9}$ | 등비급수 공식: $\sum_{k=0}^{\infty} 0.9^k = \frac{1}{1-0.9}$ |

> **수식의 의미**: 항상 오른쪽으로만 이동하는 정책을 따르면, L1에서 첫 스텝에 사과로 +1을 받지만 이후 L2에서 계속 오른쪽 벽에 부딪혀 -1씩 받는다. 할인된 무한 합으로 계산하면 -8이 된다.

---

### 수식 8-2. 등비급수 공식 (가치 계산에 활용)

$$\sum_{k=0}^{\infty} \gamma^k = \frac{1}{1 - \gamma} \quad (0 \leq \gamma < 1)$$

| 기호 | 의미 |
|------|------|
| $\sum_{k=0}^{\infty} \gamma^k$ | 할인율 $\gamma$의 무한 등비급수 |
| $\frac{1}{1-\gamma}$ | 수렴값 |

> **수식의 의미**: 할인율 $\gamma < 1$일 때, 무한히 반복되는 보상의 합이 유한한 값으로 수렴한다. 이 공식으로 Continuing task에서 무한 반복 패턴의 Return을 닫힌 형식으로 계산할 수 있다.

## 수식 전체 요약표

| 번호 | 수식 | 의미 |
|------|------|------|
| 1-1 | $S_t \in \mathcal{S}$ | 상태: 상태 집합 S의 원소 |
| 1-2 | $A_t \in \mathcal{A}(S_t)$ | 행동: 현재 상태에서 가능한 행동 집합의 원소 |
| 1-3 | $R_{t+1} \in \mathcal{R} \subset \mathbb{R}$ | 보상: 실수 집합의 부분집합 원소 |
| 1-4 | $\pi_t(a\|s)$ | 정책: 상태 s에서 행동 a를 선택할 확률 |
| 2-1 | $G_t = \sum R_{t+k}$ | Episodic Return: 보상의 단순 합 |
| 2-2 | $G_t = \sum \gamma^k R_{t+k+1}$ | Continuing Return: 할인된 보상의 합 |
| 2-3 | $G_t = R_{t+1} + \gamma G_{t+1}$ | Return의 재귀 관계 (Bellman 구조) |
| 3-1 | $\Pr\{S_{t+1}\|S_t\} = \Pr\{S_{t+1}\|S_0,\ldots,S_t\}$ | Markov 성질: 현재만으로 미래 결정 |
| 3-2 | $p(s',r\|s,a)$ | RL에서 Markov 성질 = 전이 다이나믹스 |
| 4-1 | $p(s',r\|s,a) = \Pr\{S_{t+1}=s', R_{t+1}=r \| S_t=s, A_t=a\}$ | 전이 다이나믹스: 결합 확률 |
| 4-2 | $p(s'\|s,a) = \sum_r p(s',r\|s,a)$ | 상태 전이 확률 (보상 marginalize) |
| 4-3 | $r(s,a) = \mathbb{E}[R_{t+1}\|s,a]$ | 상태-행동 쌍의 기대 보상 |
| 4-4 | $r(s,a,s') = \mathbb{E}[R_{t+1}\|s,a,s']$ | 상태-행동-다음상태 트리플의 기대 보상 |
| 5-1 | $v_\pi(s) = \mathbb{E}_\pi[G_t\|S_t=s]$ | 상태 가치 함수 |
| 5-2 | $q_\pi(s,a) = \mathbb{E}_\pi[G_t\|S_t=s, A_t=a]$ | 행동 가치 함수 |
| 6-1 | $v_*(s) = \max_\pi v_\pi(s)$ | 최적 상태 가치 함수 |
| 6-2 | $q_*(s,a) = \max_\pi q_\pi(s,a)$ | 최적 행동 가치 함수 |
| 6-3 | $\pi_*(a\|s) > 0$ only where $q_*$ max | 최적 정책 (확률적) |
| 6-4 | $\pi_*(a\|s) = 1$ if $a = \arg\max q_*$ | 최적 정책 (결정론적) |
| 6-5 | $\mu_*(s) = \arg\max_a q_*(s,a)$ | 최적 행동 |
| 7-1 | $\mathbb{E}[r] = \sum_x \sum_y p(x)p(y\|x)r(x,y)$ | 기댓값의 정의 |
| 8-1 | 등비급수 공식 $\frac{1}{1-\gamma}$ | Continuing task Return 계산 |
